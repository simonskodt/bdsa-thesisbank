# ThesisBank

> ThesisBank project in BSDA 3. semester. 

Starting from the Project bank description which involves teachers and students. Students find a thesis to apply, and teachers confirm or reject the given student.

<p>

  **OPEN IN WEBBROWSER**
  ``` 
  localhost:5077
  ```
